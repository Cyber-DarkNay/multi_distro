## 🔥 About

Cyber DarkNay Multi Distro Tool adalah tools berbasis bash yang dirancang untuk mempermudah install dan mengelola berbagai distro Linux langsung di Termux.

Tanpa ribet. Tanpa command panjang.
Semua cukup lewat menu interaktif.

---

## ⚡ Features
```
- 🧠 Simple & clean interface
- 📦 Install berbagai distro Linux
- 🚀 Login distro cepat
- 📋 List distro terinstall
- ❌ Remove distro dengan mudah
- 🎨 CLI UI keren & ringan
- 🔒 Stabil & anti error (anti blank screen)
```
---

---

### 📥 Installation
```
pkg update && pkg upgrade -y
pkg install git -y

git clone https://github.com/Cyber-DarkNay/multi_distro.git
cd multi_distro

chmod +x darknay-distro.sh
bash distro.sh
```
---

## ▶️ Usage
```
Jalankan tools:

./darknay-distro.sh

Menu utama:

[1] Install Distro
[2] Masuk Distro
[3] List Distro
[4] Hapus Distro
[5] Info System
[0] Exit
```
