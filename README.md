🔥 Cyber DarkNay Multi Distro Tool

«Powerful & Simple Multi Distro Installer for Termux using Proot-Distro
Created with ❤️ by Cyber DarkNay»

---

🚀 Overview

Cyber DarkNay Multi Distro Tool adalah script bash untuk memudahkan kamu install, menjalankan, dan mengelola berbagai distro Linux langsung di Termux.

Tools ini dibuat agar:

- ⚡ Simple
- 🔥 Cepat
- 🎯 Tanpa ribet command manual

---

✨ Features

- 📦 Install banyak distro (Ubuntu, Debian, Arch, dll)
- 🚀 Login distro dengan mudah
- 📋 List distro yang terinstall
- ❌ Hapus distro dengan cepat
- 🧠 Interface simple & user-friendly
- 🎨 Tampilan keren (CLI UI)

---

🖥️ Supported Distro

- Ubuntu
- Debian
- Arch Linux
- Alpine Linux
- Fedora
- Manjaro
- OpenSUSE
- Deepin
- AlmaLinux
- Rocky Linux
- Void Linux

---

📥 Installation

pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/username/darknay-distro
cd darknay-distro
chmod +x darknay-distro.sh
./darknay-distro.sh

---

▶️ Usage

Jalankan tools:

./darknay-distro.sh

Pilih menu sesuai kebutuhan:

- Install distro
- Login distro
- Hapus distro
- dll
